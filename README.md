# Rest-Blog-Ui
React User Interface Blog 

A blog app made using Django Rest Framework, Django Rest Auth, React, React Hooks, Axios, React Markdown and Semantic-UI. This application allows users to view the latest post on the blog. However, users are restricted on what they are able to do, depending on permissions and seeing whether they're authenticated. Users are required to signup and login to either create a post or update/delete the post, depending if theyâ€™re the author of the post.
